gctimer (t, cb, self, ...)
A garbage-collectible timer implementation. Provides a timer that can be attached to an object, and GC'ed along with that object, as opposed to regular timers that keep running and prevent the object from being GC'ed.

Parameters:

  • t time in ms
  • cb callback for the timer. The signature is function(premature, self, ...)
  • self the object the timer belongs to
  • ... any additional parameters to pass to the timer callback

Returns:

    timer handle (same as ngx.timer.at)
